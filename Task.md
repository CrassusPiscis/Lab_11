P3121. Лабораторная работа 11. Проект (4)
Описание предметной области

Разработать веб-приложение «Каталог книг», в котором пользователи могут управлять списком книг.

Сущности: пользователь, книга.

Книга описывается:

    названием,

    автором,

    годом издания,

    жанром,

    кратким описанием,

    статусом чтения (не начата / читаю / прочитана).

Пользователь может:

    добавлять книги в каталог,

    редактировать сведения,

    удалять записи,

    отмечать статус чтения,

    просматривать каталог.

На главной странице — список всех пользователей с количеством книг в их каталогах.

CRUD обязателен для всех сущностей.
Технические требования

Выбранный фреймворк:
    Flask (сложность: средняя)

В приложении необходимо реализовать фронтэнд и бэкэнд составляющие. Для приложения используется архитектура MVC.

Фронтэнд представлен шаблонами на Jinja2 или с использованием другого шаблонизатора. Может использоваться либо чистый HTML, либо Bootstrap 5.

В проекте используется база данных SQLite (можно хранить данные в файле или в памяти). Рекомендуется изучить и использовать компонент ORM, который позволяет работать с БД без выполнения запросов напрямую, а через программную прослойку.

При использовании ORM правильно и хорошо использовать концепцию миграций и применять их к БД. Прочитайте про миграции для конкретного фреймворка и ORM, которая в нем используется (Flask-SQLAlchemy)

Для Flask рекомендуется Flask-SQLAlchemy.
